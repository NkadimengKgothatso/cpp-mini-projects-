🚀 C++ Problem Solving Projects

I recently worked on a collection of small C++ programs focused on practicing problem-solving, algorithmic thinking, and basic programming logic. Each program solves a specific computational problem and demonstrates the use of loops, conditionals, functions, and string/number manipulation.

📂 Projects Included

🔹 Amicable Numbers Checker  
A program that determines whether two numbers are amicable numbers. Amicable numbers are two different numbers where the sum of the proper divisors of each number equals the other number.

🔹 Case Dominance Checker  
This program analyzes a string and determines whether uppercase or lowercase letters dominate in the input.

🔹 Second Largest Number Finder  
A C++ program that takes a list of numbers and finds the second largest value without sorting the entire list.

🔹 Vowel–Consonant Difference Calculator  
A program that counts vowels and consonants in a string and calculates the difference between them.

💡 Skills Demonstrated
- C++ programming fundamentals  
- Problem solving and algorithm design  
- Conditional logic and loops  
- String and number manipulation  
- Clean and readable code structure  

🎯 These exercises helped strengthen my understanding of C++ logic building and algorithmic thinking, which are essential skills for software development and computer science.
